# 01 — 全局视觉语言规范

> **Step 6.1** | 适用范围：系统所有界面层
>
> 后端依赖：无（纯前端规范）

---

## 1. 设计哲学四原则

| 原则 | 描述 | 强制等级 |
|------|------|---------|
| **结构优先可视化** | 界面必须优先展示底层数据结构（剧情拓扑、角色状态矩阵），而非空白文本编辑器 | 必须 |
| **结构化 I/O 脚手架** | 配合后端 JSON 数据流，使用表单/滑块/下拉/分页 Prompt 面板，最小化自由文本 | 必须 |
| **单智能体专注模式** | 每个 Agent（Planner/Writer/Critic/Editor）必须有独立展示层与明确交互边界 | 必须 |
| **全链路可观测性** | AI 决策/工具调用/Token 消耗/重试全部以结构化日志+时间轴可视化呈现 | 必须 |

---

## 2. 色彩系统

### 2.1 基础色阶（暗黑模式 · 7 层）

| 层级 | Hex | RGB | 用途 |
|------|-----|-----|------|
| **基础深空背景** `Base Space` | `#0a0a0b` | (10, 10, 11) | 全局主背景；消除屏幕边界感，视觉焦点聚向前景 |
| **主承载面** `Surface L1` | `#1b1c22` | (27, 28, 34) | 主要内容卡片、大型模态框、侧边栏导航 |
| **交互浮层** `Surface L2` | `#2d2f3a` | (45, 47, 58) | Hover 态、激活输入框、下拉菜单、选中行 |
| **主文本** `Primary Text` | `#f2efe9` | (242, 239, 233) | 正文、主标题、AI 生成核心文本；暖象牙色防眩光 |
| **辅助文本** `Secondary Text` | `#999999` | (153, 153, 153) | 占位符、说明文字、时间戳、非激活标签 |
| **AI 活跃态** `AI Active` | `#2ef2ff` | (46, 242, 255) | LLM 推理中 / 工具调用中 / 任务排队中 |
| **人工干预** `Human-in-the-Loop` | `#ff2e88` | (255, 46, 136) | HITL 审批请求 / 剧情严重冲突 / 角色行为失控 |
| **成功/记忆锚定** `Success / Memory` | `#3f5a48` | (63, 90, 72) | 数据已存入向量库 / 记忆精准命中 / 一致性校验通过 |

> ⚠️ **禁止**使用纯黑 `#000000` 或纯白 `#ffffff` 作为背景或正文色。

### 2.2 语义状态色（叠加使用）

| 状态 | 颜色 | 不透明度 | 说明 |
|------|------|---------|------|
| NodeStatus.ACTIVE | `#2ef2ff` | 60% 描边 | 当前正在执行的剧情节点 |
| NodeStatus.COMPLETED | `#3f5a48` | 80% 填充 | 已完成节点 |
| NodeStatus.SKIPPED | `#999999` | 40% 填充 | 跳过/剪枝节点 |
| NodeStatus.PENDING | `#2d2f3a` | 100% 填充 | 待执行节点 |
| 轻微设定冲突 | `#f5a623` | 虚线底边 | 琥珀色警告线 |
| 严重 OOC 违规 | `#ff2e88` | 波浪底边 | 猩红波浪线 |

## 3. 排版规范

### 3.1 字体层级

| 层级 | 字号 | 字重 | 行高 | 用途 |
|------|------|------|------|------|
| H1 主标题 | 24px | 700 | 1.3 | 页面/模块大标题 |
| H2 小节标题 | 18px | 600 | 1.4 | 卡片标题、面板标题 |
| H3 标签 | 14px | 500 | 1.5 | 属性标签、徽章文字 |
| Body 正文 | 15px | 400 | 1.7 | 生成文本、说明文字 |
| Caption 注释 | 12px | 400 | 1.6 | 时间戳、Token 数值 |
| Code 代码 | 13px (等宽) | 400 | 1.5 | JSON 数据、日志流 |

### 3.2 文本阅读列宽

- **AI 生成文本区域**：最大宽度 `65ch`（约 75 字符），居中或左对齐
- **数据可视化标签**：不限宽，但单行不超过视口 `40%`
- **日志流/代码块**：全宽，启用横向滚动

## 5. 微交互动效规范

### 5.1 AI 思考状态（呼吸灯效）

**禁止**使用无意义旋转 Spinner。AI 推理时必须呈现：

```
组件外框：  #2ef2ff 描边，opacity 在 30%–80% 之间往复（周期 2s，ease-in-out）
底部日志：  模糊滚动文字流（font-size 11px，opacity 50%，blur 1px）
           内容示例："调用 WriterAgent.write() → 上下文注入 2840 tokens → 生成中..."
```

### 5.2 状态转换动画

| 场景 | 动画 | 时长 |
|------|------|------|
| 侧边栏推出 | slide-in-right | 200ms ease-out |
| 模态框出现 | fade-in + scale 0.96→1.0 | 180ms ease-out |
| Toast 通知 | slide-in-top + fade | 160ms |
| 节点卡片高亮 | border-color transition | 300ms |
| Diff 展开 | height expand + fade-in | 250ms |
| HITL 遮罩 | backdrop fade-in | 120ms |

### 5.3 粒子效果（打回循环）

Agent 打回工单时：从 Critic 泳道指向 Writer 泳道画一条带红色流动粒子的逆向虚线。
- 粒子颜色：`#ff2e88`
- 粒子大小：3px 圆点
- 动画速度：沿贝塞尔曲线匀速 600ms

### 5.4 打字机效果（文本流）

生成文本逐字呈现时：
- 速度：约 35ms / 字（可根据用户偏好加速）
- 文本区背景边界：呼吸灯 `#2ef2ff` 光晕（intensity: 0.4）
- 光标：1px 实线，`#2ef2ff`，0.5s 闪烁

## 6. 图标规范

- **风格**：单色、可变线宽的线性矢量图形（Line Icons）
- **尺寸**：16px / 20px / 24px（三档）
- **颜色**：继承父级文本色（`#f2efe9` 或 `#999999`）
- **语义图标**（固定不可改）：
  - 剧情节点：拓扑网络图标
  - 角色：人形轮廓
  - 记忆：脑/时钟叠加
  - 技能：积木/拼图
  - LLM 调用：闪电
  - 警告：感叹号三角
  - HITL 审批：挂锁
  - 成功：勾形
  - 插件：电插头

---

## 7. 交互焦点与无障碍

- 键盘焦点环：`#2ef2ff` 2px 实线，`outline-offset: 2px`
- 最小点击区域：44×44px
- 颜色对比度：主文本 `#f2efe9` 对底色 `#0a0a0b`，对比度 ≥ 11:1（AAA 级）
- 辅助文本 `#999999` 对 `#1b1c22`，对比度 ≥ 4.5:1（AA 级）
