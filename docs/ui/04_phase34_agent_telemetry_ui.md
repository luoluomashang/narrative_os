#### 工单卡片规格

```
┌── 工单卡片（#2d2f3a · 圆角 6px · 宽 160px）─┐

```

工单状态颜色：
- 排队中：边框 `#999999`
- 执行中：边框 `#2ef2ff` 呼吸闪烁
- 打回中：逆向粒子线 `#ff2e88`
- 完成：填充 `#3f5a48` 20%
- 失败：边框 `#ff4040`

### 1.2 打回动画规范

```
触发条件：CriticAgent 判定未通过质量阈值
          （CriticReport.passed == False）

动画：
  ① Critic 泳道 → Writer 泳道 画一条逆向贝塞尔曲线
  ② 曲线上：红色粒子 (直径 3px · #ff2e88) 沿曲线匀速流动 600ms
  ③ 工单卡片回到 Writer 位置，标题前追加 "[重写#2]"
  ④ 右上角 Toast：
     "Critic Agent 判定未通过，已发起第 2 次重写请求"

硬性上限（来自降级策略配置）：
  重试次数 > 配置阈值（默认 3）→ 自动中止，弹出 HITL 审批模态框通知用户
```

