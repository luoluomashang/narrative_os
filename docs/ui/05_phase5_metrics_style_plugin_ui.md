## 2. 风格引擎控制台（Style Engine Interface）

**后端映射**：`skills/style_engine.py` → `StyleEngine`

### 2.1 控制台布局

```
┌── 风格控制台（独立面板，可浮动或嵌入右侧面板）─────────────────────┐
│  文风指纹（Style Fingerprint）                                    │
│  ─────────────────────────────────────────────────────────────  │
│  形容词/副词密度        [────●──────]   低密度 → 高密度  (0–100)   │
│  句式长度与复杂度        [──────●────]   短促 → 繁复     (0–100)   │
│  对话与动作行文比例       [─●──────────]  对话重 → 动作重 (0–100)   │
│  视点凝视深度            [────────●──]   旁观 → 沉浸      (0–100)  │
│  诗化意象密度            [───●───────]   写实 → 意象浓厚  (0–100)  │
│                                                                  │
│  ─── 样书锚定区 ─────────────────────────────────────────────── │
│  ┌─────────────────────────────────────────┐                    │
│  │  拖拽参考作者文本至此（.txt / .md）        │                    │
│  │  或从样书库选择：                         │                    │
│  │  [海明威·极简] [吉布森·赛博] [金庸·武侠]  │                    │
│  └─────────────────────────────────────────┘                    │
│  ✓ 已提取目标文风：海明威（句式短促·动作重·对话直接）              │
│    滑块已自动定位至对应参数                                        │
│                                                                  │
│  ─── 实时预览 ──────────────────────────────────────────────── │
│  [预览文本框]  按当前滑块参数生成一段示例句               [刷新]    │
│                                                                  │
│  [应用到当前章节]  [保存为风格预设]  [重置为默认]                  │
└────────────────────────────────────────────────────────────────┘
```

### 2.2 风格参数与后端映射

| 滑块 | `StyleEngine` 内部字段 | 说明 |
|------|----------------------|------|
| 形容词/副词密度 | `adjective_density` | 0=极简，100=华丽 |
| 句式长度复杂度 | `sentence_complexity` | 0=短促，100=繁复嵌套 |
| 对话与动作比例 | `dialogue_action_ratio` | 0=对话重，100=叙事重 |
| 视点凝视深度 | `pov_depth` | 0=第三人称旁观，100=深度意识流 |
| 诗化意象密度 | `imagery_density` | 0=写实，100=意象堆叠 |

### 2.3 样书锚定交互流

1. 用户拖拽文本文件至锚定区（或点击预设样书）
2. `StyleEngine` 后台提取文风参数（特征分析）
3. 5 个滑块自动动画滑动至新位置（transition 500ms）
4. 界面反馈："已成功提取并覆写目标文风特征"（Toast · `#3f5a48`）
5. 用户可在此基础上手动微调滑块

### 2.4 风格预设库

```
预设库下拉（常驻于控制台顶部）：
  [武侠·金庸]  [都市·硬派]  [奇幻·史诗]  [恐怖·压抑]  [+ 自定义]

选择预设 → 滑块批量更新（动画过渡）→ 同步到 StyleEngine
```

### 3.2 热拔插交互流

1. 用户在卡片上拨动 **Toggle 开关**（ON→OFF 或 OFF→ON）
2. 调用 `PluginRegistry.enable(plugin_id)` / `.disable(plugin_id)`
3. 无需重启，新技能卡片**即时注入**到 Phase 1 的技能积木库（动画：从集市卡片飞出，降落到技能库）
4. 场景生成器中，相关 Agent 可立即识别并调用
5. Toast 通知（`#3f5a48`）："悬疑手法漏洞扫描器 已启用并注入技能积木库"

### 3.4 PluginRegistry 与 UI 状态同步

| `PluginRegistry` 方法 | 触发时机 | UI 反馈 |
|-----------------------|---------|---------|
| `list_plugins()` | 集市页面加载 | 渲染瀑布流 |
| `enable(plugin_id)` | 用户拨动 Toggle ON | 卡片变激活态 + 飞入技能库动画 |
| `disable(plugin_id)` | 用户拨动 Toggle OFF | 卡片变灰 + 从技能库移除 |
| `get_plugin(plugin_id)` | 点击查看详情 | 展开详情抽屉 |
