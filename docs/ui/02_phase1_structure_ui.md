## 1. 剧情图谱无限画布（Plot Graph Canvas）

**后端映射**：`core/plot.py` → `PlotGraph`, `PlotNode`, `NodeStatus`，API：`POST /chapters/plan`

### 1.1 画布布局

```
┌─────────────────────────────────────────────────────────────┐
│ 工具栏（新建节点 / 连线 / 张力视图切换 / 缩放 / 全局搜索）         │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│          [像素点阵网格背景 · dot size 1px · opacity 15%]       │
│                                                             │
│   ┌──────────────┐     贝塞尔曲线+箭头       ┌────────────┐  │
│   │ 节点卡片 A   │ ──────────────────────→  │ 节点卡片 B │  │
│   └──────────────┘                          └────────────┘  │
│                                                             │
└─────────────────────────────────────────────────────────────┘
         ↑ 双击节点 → 右侧属性面板推出
```

### 1.2 节点卡片规格

```
┌─── PlotNode 卡片（#1b1c22 · 圆角 8px · 宽 240px）──────────┐
│ [编号徽章] ch03_001   [NodeStatus 灯：●ACTIVE #2ef2ff]    │
│ ─────────────────────────────────────────────────────── │
│  标题：（主标题·H2·最多 2 行）                               │
│  概要：（最多 80 字·Body·#999999）                          │
│ ─────────────────────────────────────────────────────── │
│ [POV角色头像 20px] [地点标签] [张力值 7/10] [时间戳]         │
│ [关键节拍 🔴] ← 启用时显示 #ff2e88 霓虹描边                  │
└────────────────────────────────────────────────────────┘
```

**NodeStatus 颜色映射**（来自 `core/plot.py · NodeStatus`）：

| `NodeStatus` 值 | 卡片视觉 |
|----------------|---------|
| `PENDING` | 边框 `#2d2f3a`，无光晕 |
| `ACTIVE` | 边框 `#2ef2ff` 60% + 外发光 |
| `COMPLETED` | 背景填充 `#3f5a48` 20% + 对勾徽章 |
| `SKIPPED` | 整体 opacity 40%，斜线遮罩 |

### 1.3 节点属性编辑面板（右侧推出）

触发条件：双击任意节点卡片

### 1.4 张力波形视图

工具栏 **"张力视图"** 切换后：

- 画布下方展开一条横向波形曲线（宽度 = 画布全宽）
- 纵轴：0–10 张力值
- 曲线颜色：按张力值从左到右渐变（0=冷蓝 `#2ef2ff`，10=警示红 `#ff4040`）
- 平坦区（连续节点差异 < 1.5）显示黄色感叹号 ⚠ + 提示卡：  
  "当前段落缺乏节奏变化，建议调用 Planner Agent 插入次要冲突"

## 2. 角色状态矩阵（Character State System）

**后端映射**：`core/character.py` → `CharacterState`, `BehaviorConstraint`, `VoiceFingerprint`

### 2.1 角色档案多标签页

```
┌── 角色全息档案（tabbed navigation）──────────────────────────┐
│  [基础属性 Profile] | [动态状态 State] | [行为约束 Constraints] │
│  [声音指纹 Voice]   | [关系网络 Relations]                     │
└────────────────────────────────────────────────────────────┘
```

#### Tab 1：基础属性 Profile

- 头像上传区（圆形，80px；拖拽或点击上传）
- 姓名、别称（文本框）
- 阵营 / 核心动机（文本框）
- 人物简介（多行，上限 300 字）

#### Tab 2：动态状态 State（来自 `CharacterState`）

```
状态可视化：雷达图（6轴）+ 多色进度条

雷达图轴：心理承受力 / 道德倾向 / 体力 / 理智值 / 意志力 / 情感稳定性
                         （0–100 · 实时数值来自 CharacterState 字段）

差异对比视图（Diff State View）：
  [时间滑块 ← 初始基准 ────●──── 当前节点 →]
  左侧：初始雷达图（灰色半透明）
  右侧 / 叠加：当前雷达图（彩色实线）
  变化幅度 > 20 的轴：显示浮动数字 "+23" 或 "-15"
```

#### Tab 3：行为约束 Constraints（来自 `BehaviorConstraint`）

**Visual Rule Builder（可视化规则生成器）**：

```
规则块 1：  IF [角色生命值] < [20%]  THEN [强制触发 '撤退' 倾向]
            [字段下拉] [运算符] [阈值] → [行为标签]

规则块 2：  NEVER [向敌对阵营妥协]
            [NEVER 徽章·红色] + [行为描述文本]

已生成规则 → 带挂锁图标的硬性约束标签
（传给 CriticAgent 作为 OOC 判定刚性标准）
```

#### Tab 4：声音指纹 Voice（来自 `VoiceFingerprint`）

- 常用词频（标签云，字号对应频率）
- 句式均匀度滑块（只读 · 由 StyleEngine 自动填充）
- 口头禅列表（可编辑标签组）

#### Tab 5：关系网络 Relations

→ 见 2.2

### 2.2 力导向关系网络图

```
全局关系图（force-directed graph）
  节点：角色头像圆形节点（直径 48px）
  连线：
    结盟/友善 → 绿色实线 (#3f5a48)，线宽对应亲密度
    敌对/仇恨 → 红色锯齿虚线 (#ff4040)
    利益关系  → 橙色点线 (#f5a623)
    未知/模糊 → 灰色细线 (#999999)

底部：[全局时间线滑块 ← 序章 ──────●── 当前章节 → 结局]
随滑块拖动：连线颜色/线宽/关系标签实时更新（动画 300ms 过渡）
```

## 3. 记忆系统三层界面（Memory & Lore）

**后端映射**：`core/memory.py` → `MemorySystem`, `MemoryRecord`

### 3.1 三层抽屉面板

界面右侧或底部的抽屉，纵向分三区：

```
┌── 短期记忆区 Short-term ──────────────────────────────┐
│  时间线信息流（向上滚动）                                 │
│  ● ch05_003 · 白明远决定分道扬镳 · 2min ago             │
│  ● ch05_002 · 剑阵破防，紫气外泄 · 5min ago             │
│  [活跃指示：#2ef2ff 左边框]                              │
├── 中期记忆区 Mid-term ────────────────────────────────┤
│  卡片网格（2列）                                        │
│  ┌──────────┐ ┌──────────┐                           │
│  │ ch04节点 │ │ ch03高潮 │  [来自 MemoryRecord.layer=mid] │
│  └──────────┘ └──────────┘                           │
├── 长期记忆区 Long-term / Lore ────────────────────────┤
│  星空图可视化（SVG 力导向散点）                           │
```

## 4. 技能积木库（Skill DSL）

**后端映射**：`skills/dsl.py` → `SkillRegistry`, `SkillSpec`

### 4.1 技能卡片规格

```
┌── 技能卡片（#1b1c22 · 宽 280px）─────────────────────────┐
│  [图标] 恐怖氛围渲染                    [已启用 ●]         │
│  描述：基于环境描写强化恐惧情绪 (14 字描述)                  │
│ ─────────────────────────────────────────────────────  │
│  情感倾向   [─────●──────]  中性 → 恐惧  (0–10)           │
│  句式长短   [──●───────]   短促 → 冗长   (0–10)           │
│ ─────────────────────────────────────────────────────  │
│  填空通道：在[___场景___]中，渲染[___情绪___]              │
│ ─────────────────────────────────────────────────────  │
│  参考资料池：[拖拽设定文档至此区域]                         │
│ ─────────────────────────────────────────────────────  │
│  [添加到工作流]  [查看 SkillSpec JSON]                    │
└───────────────────────────────────────────────────────┘
```

### 4.2 技能库面板

- 布局：卡片瀑布流，左侧有分类过滤器（恐怖/悬疑/奇幻/武侠/言情 等）
- 来源：`SkillRegistry.list_skills()` 动态拉取
- 搜索框：关键词实时过滤
- **"添加到工作流"**：拖拽卡片至场景生成器中栏的工作流待执行队列

### 4.3 参数控件类型总表

| 控件类型 | 适用参数场景 | 后端字段 |
|---------|------------|---------|
| 水平滑块 (0–10) | 情感倾向、句式长短、创意温度 | `SkillSpec.params` |
| 下拉菜单 | 类型选择、模型路由 | `SkillSpec.params` |
| 填空模板 | 自然语言任务描述 | `SkillSpec.cloze_template` |
| 拖拽文件区 | 参考资料注入 | `SkillSpec.reference_materials` |
| 多选标签 | 角色约束、世界规则 | `SkillSpec.constraints` |
