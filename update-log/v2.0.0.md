# Narrative OS v2.0.0 更新日志

- 发布日期: 2026-04-11
- 版本基线: v2.0.0

## 新增

- 新增独立页面:
  - `/project/:id/memory-search` 记忆检索页面
  - `/project/:id/trace` 执行链路检视页面
- 侧边栏补齐入口:
  - 数据管理新增“记忆检索”入口
  - 质量工具新增“执行链路”入口

## 优化

- Metrics 仪表盘改为真实数据优先:
  - 移除空项目固定示例曲线
  - 空数据时显示真实空态与 0 统计
- Agent 工坊改为真实 Trace 驱动:
  - 移除 mock Agent 卡片与固定演示章节
  - 无链路数据时显示空态
- Character Matrix 稳定性优化:
  - Drive/Social 页移除运行时模板字符串组件
  - 消除 Vue runtime 编译告警
- Trace 展示文案优化:
  - 将技术占位文本 `tracing not yet available` 映射为用户可理解空态提示

## 废弃与清理

- 废弃 Metrics/Agent 页面中的演示数据 fallback 逻辑
- 清理使用手册中的错位章节、重复段落与无效编号

- 本次更新为增量升级，不影响现有项目数据结构与核心 API。
- 对无 Trace 数据、无历史 Metrics 数据的项目，界面将显示空态，不再显示示例演示数据。
